# Dispatcher-Assist-AI-System
Overview

Dispatcher-Assist AI System is an AI-assisted emergency dispatch support system designed to help emergency call centers analyze incoming call information more efficiently. The system aims to classify emergency types, extract key information from call descriptions, and support location mapping for faster decision-making.

This project was developed as a final project to explore the use of artificial intelligence, natural language processing, and speech-to-text technologies in emergency response support.

Features
Emergency type classification from text input
Speech-to-text integration for call transcription
Location keyword extraction and mapping support
Real-time call analysis concept
Simple interface design for dispatcher assistance
Support for future bilingual Thai-English operation
Technologies Used
Python
Natural Language Processing (NLP)
Speech-to-Text
Machine Learning
Data Processing
Location Mapping
Streamlit / Web Interface Concept
System Workflow
User or dispatcher inputs emergency call information.
The system converts speech or text into structured information.
NLP processing extracts important keywords such as emergency type, location, and severity.
The classifier predicts the emergency category.
The system displays suggested information to support dispatcher decision-making.
